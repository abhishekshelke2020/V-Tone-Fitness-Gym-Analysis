# V Tone Fitness Gym Analysis | Power BI Dashboard

This project presents a comprehensive analysis of membership and workout data for V Tone Fitness Gym, Chicago. The objective is to evaluate member engagement, demographic trends, and overall fitness performance through an interactive Power BI dashboard.

The dashboard delivers clear, data-driven insights into key performance indicators, including:

Total Members: 68.97K

Active vs Inactive Member Distribution

Average Workouts per Week

Average Workout Duration

Total Weight Loss Achieved

Age Distribution Analysis

Gender-Based Membership Comparison
